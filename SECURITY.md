# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Reporting a Vulnerability

If you discover a security issue in Propaga, please report it privately:

1. Open a [GitHub Security Advisory](https://github.com/hocestnonsatis/propaga/security/advisories/new), or
2. Email the maintainer at **anil.oz@icloud.com** with a description of the issue and steps to reproduce.

Please do not file public issues for undisclosed security vulnerabilities.

We aim to acknowledge reports within 72 hours and will work with you on a fix and coordinated disclosure timeline.
